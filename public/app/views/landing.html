The Wizards Unicorn

Welcome to the Greatest Adventure of your life!

This is a VOICE controlled text adventure game. You are the best 3rd class Wizard of your time. Make your way through the world, collecting the items you need to reach far flung destinations and discover...

...ATLANTIS!


Getting Started: Sign up or login to get access to the game. Once signed in click the 'Adventure' option from the menu above. The game page will show and instructions on voice control will be available.

Technology: This site was built using the MEAN stack (MongoDB, Express, Angular.JS and Node.JS) along with JavaScript, Bootstrap and Parallax

API: We used Microsoft's Voice to Text API released during BUILD 2016. The API records 5 second of voice and uses their cloud AI platform to convert it to text which is then used as an input for the game. The text adventure was created using Angular and JavaScript and is an original piece of work by Ethan Young. The game elements are held on MongoDB which is accessed throughout the adventure to save the state of the game and record progress.
